# Алгоритмы и структуры данных

Любое собеседование или отбор на стажировку включает в себя алгоритмическую секцию.
Чтобы успешно к ней подготовиться, нужна постоянная практика.

Лучше всего делать это на [LeetCode](https://leetcode.com/). 
Подборки находятся в конце.

Также у яндекс практикума есть бесплатный [курс по алгоритмическим собеседованиям](https://start.practicum.yandex/algorithms-interview/).
